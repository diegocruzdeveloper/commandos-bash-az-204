Guia Rápido de Comandos Azure CLI

Resumo para a Certificação AZ-204

Azure App Service

Ação Comando Resumido
Criar Plano az appservice plan create -g <rg> -n <plan> --sku B1 --is-linux
Criar Web App az webapp create -g <rg> -p <plan> -n <app> --runtime "DOTNET|8.0"
Definir App Settings az webapp config appsettings set -g <rg> -n <app> --settings "K=V"

Soluções de Contêineres (ACR, ACI, ACA)

Ação Comando Resumido
Criar ACR az acr create -g <rg> -n <acr_name> --sku Basic
Criar ACI az container create -g <rg> -n <container> --image <image> --dns-name-label <dns>
Criar App Contêiner az containerapp create -g <rg> -n <aca_app> --environment <env> --image <image>

Azure Functions

Ação Comando Resumido
Criar Conta Storage az storage account create -g <rg> -n <storage> --sku Standard_LRS
Criar Function App az functionapp create -g <rg> -n <func_app> --storage-account <storage> --consumption-plan-location <loc>
Definir App Settings az functionapp config appsettings set -g <rg> -n <func_app> --settings "K=V"

Azure Blob Storage / Storage Queue

Ação Comando Resumido
Criar Contêiner Blob az storage container create -n <container> --account-name <storage>
Fazer Upload de Blob az storage blob upload -c <container> -f <file> -n <blob_name> --account-name <storage>
Criar Fila (Queue) az storage queue create -n <queue> --account-name <storage>
Enviar Mensagem Fila az storage message put -q <queue> --content "Hello" --account-name <storage>

Azure Cosmos DB (SQL API)

Ação Comando Resumido
Criar Conta az cosmosdb create -g <rg> -n <cosmos_account>
Criar Banco de Dados az cosmosdb sql database create -g <rg> -a <cosmos_account> -n <db_name>
Criar Contêiner az cosmosdb sql container create -g <rg> -a <cosmos_account> -d <db_name> -n <container> -p /pk

Entra ID / Key Vault / Microsoft Graph

Ação Comando Resumido
Criar Registro de App az ad app create --display-name "MyApp"
Criar Service Principal az ad sp create --id <app_id>
Criar Key Vault az keyvault create -g <rg> -n <kv_name>
Definir Segredo az keyvault secret set --vault-name <kv_name> -n <secret_name> --value <secret>
Definir Política de Acesso az keyvault set-policy -n <kv_name> --object-id <obj_id> --secret-permissions get

Service Bus / Event Grid / Event Hub

Ação Comando Resumido
Criar Namespace SB az servicebus namespace create -g <rg> -n <sb_namespace>
Criar Fila SB az servicebus queue create -g <rg> --namespace-name <sb_namespace> -n <queue>
Criar Tópico Event Grid az eventgrid topic create -g <rg> -n <eg_topic> -l <location>
Criar Namespace EH az eventhubs namespace create -g <rg> -n <eh_namespace>
Criar Event Hub az eventhubs eventhub create -g <rg> --namespace-name <eh_namespace> -n <hub_name>

API Management (APIM)

Ação Comando Resumido
Criar Instância APIM az apim create -g <rg> -n <apim_name> --publisher-name "Org" --publisher-email email@... --sku-name Consumption
Importar API az apim api import -g <rg> -n <apim_name> --path /myapi --api-id myapi --specification-url <url>
